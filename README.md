Plant Disease Detection using Image Processing
This project detects diseases in Maize, Potato, and Chilli plants using image processing and deep learning.
The system allows users to upload a leaf image and automatically predicts the disease.
After detecting the disease, the system also displays possible remedies and treatment suggestions to help farmers take proper action.


Features:
-Upload plant leaf image
-Detect plant disease using deep learning
-Supports Maize, Potato, and Chilli plants
-Displays disease name
-Shows remedies and treatment suggestions
-Simple and user-friendly web interface.

Technologies Used:
-Python
-PyTorch
-ResNet18
-Flask
-HTML
-CSS
-Image Processing


How to Run the Project?

Install required libraries
-pip install -r requirements.txt

Run the application
-python app.py

Open browser and go to
-http://127.0.0.1:5000

Upload a plant leaf image to detect disease and view remedies.
